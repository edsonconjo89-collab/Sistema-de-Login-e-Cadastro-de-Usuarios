Sistema de Login e Gestão de Usuários (Java Swing)

Aplicação desktop desenvolvida em Java com Swing, com foco na prática de Programação Orientada a Objetos (POO).

Este é o meu primeiro projeto unindo POO + Swing, desenvolvido como parte do meu processo de aprendizagem em Engenharia Informática.


---

Funcionalidades

Sistema de login

Diferenciação básica de usuários (ex.: administrador)

Gestão de usuários

Listagem

Atualização de dados


Tela “Meu Perfil”

Visualização dos dados do usuário logado

Secção “Sobre”


Interface gráfica criada com Swing (NetBeans GUI Builder)


Funcionalidades em desenvolvimento

Centralização do usuário logado

Alteração de senha

Remoção e edição de usuários via interface

Validações adicionais e melhorias de robustez
